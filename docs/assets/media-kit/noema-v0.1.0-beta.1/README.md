# Noema 0.1 Beta 1 媒体发布素材包

状态：公开媒体素材归档
Release：`v0.1.0-beta.1`
仓库：`MrPPFruit/noema`

这个目录归档 Noema 首次公开 beta 可对外复用的图片和视频素材，适用于 GitHub、release notes、社媒发布、商店准备和产品介绍。

## 目录说明

| 目录 | 用途 |
|---|---|
| `github/` | GitHub README、仓库卡片、release 宣传图 |
| `screenshots/` | 功能截图和产品细节图 |
| `store/` | 商店风格截图序列和 feature graphic |
| `brand/` | App 图标媒体图 |
| `video/` | 产品预览短视频 |

## 发布规则

- 这些素材是 Noema 官方展示素材，不是示例照片包。
- 可见照片来自 App 截图或截图排版，不是可复用的公开照片数据集。
- App 图标、截图、feature graphic 和视频不作为 MPL-2.0 可复用资产；品牌与商标边界仍然适用。
- 后续公开版本如果改变产品可见行为，应创建新的媒体包版本目录，不要静默覆盖这个归档。
