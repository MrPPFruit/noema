## 变更

- 新增 `鉴` 页面长按选择、删除和批量删除照片。
- 新增照片详情 sheet 中的单张删除入口。
- 删除确认 UI 与 Noema 既有删除弹窗保持一致。
- Android 删除支持仅从 Noema 移除，或在用户和系统确认后同时删除系统相册原图。
- 修正 Android release 构建签名配置，本地 release keystore 存在时使用官方 release 证书。

## 隐私与安全

- Noema 不默认后台扫描整机图库。
- Noema 不会自动删除系统相册原照片。
- 只有用户明确选择删除原图，并且 Android 接受系统级删除请求后，才会尝试删除系统相册原图。
- AI 请求逻辑没有变化：只有用户配置 Provider 并主动触发 AI 品鉴时，照片才会发送给用户选择的 Provider。
