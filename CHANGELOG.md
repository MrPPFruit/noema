## 首次公开 release 默认策略

首次公开 release 默认采用：

- tag：`v0.1.0-beta.1`。
- App version：`0.1.0-beta.1+11`。
- release 类型：GitHub pre-release。
- APK：建议上传 release-signed APK。
- SHA256：提供 `SHA256SUMS.txt`。

发布前仍必须最后确认：

- 公开仓库 URL。
- 是否需要 signed tag。
- release notes 与 APK 文件名。
