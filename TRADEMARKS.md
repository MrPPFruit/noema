# 品牌与商标边界

状态：当前公开仓库说明。

## 原则

Noema 源码采用 MPL-2.0，但 Noema 名称、Logo、图标、截图、商店素材、包名、bundle id 和发布渠道不自动随源码许可授权。

这条边界不是反开源，而是为了避免用户把非官方 fork 当成官方 App。

## 官方标识

官方项目边界：

- 官方仓库 URL。
- 官方 Android package name：`com.mrppfruit.noema`。
- 官方 iOS bundle identifier：`com.mrppfruit.noema`。
- 官方 release tag 规则。

## Fork 要求

如果发布修改版或 fork：

- 使用不同的 App 名称。
- 使用不同的图标和商店素材。
- 使用不同的 package name 和 bundle id。
- 不暗示自己是官方版本。
- 不使用官方 release notes、截图或下载链接包装成自己的发布。

## 截图和素材

公开 README、商店图、release notes 中使用的照片和截图，需要单独确认授权。私人旅行照片不能在没有项目负责人明确确认的情况下进入公开资产。
